# AI Legal Document Analyzer

## Features
- Upload legal PDF
- AI-powered analysis
- Summary + clauses + risks

## Tech Stack
- Flask
- OpenAI API
- PyPDF2

## How to Run

```bash
git clone https://github.com/yourusername/legal-analyzer
cd legal-analyzer

pip install -r requirements.txt

export OPENAI_API_KEY=your_key   # Linux/Mac
set OPENAI_API_KEY=your_key      # Windows

python app.py
